ES6 Modules

Wepack um Module importieren zu können

      npm install webpack --save-dev

      webpack src/08_ES6_modules.js output/08_ES6_modules.js
oder via webpack.config.js 
      var webpack = require('webpack');

      module.exports = {
        entry: './src/08_ES6_modules.js',
        output: {
          filename: './output/08_ES6_modules.js'
        }
      }
und nur webpack ausführen jetzt wird aber nicht in ECMA6 konvertiert, darum nutzen wir den babel-loader 
        npm install babel-loader --save-dev
und in webpack.config.js 
      module: {
        rules: [{
          test: /\.js$/,
          exclude: /(node_modules|bower_components)/,
          use: {
            loader: 'babel-loader',
            options: {
              presets: ['env']
            }
          }
        }]
      }
package.json 
      "build": "webpack"
